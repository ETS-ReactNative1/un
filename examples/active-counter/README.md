## Runing the Example

### Run in browser with no installations

```sh
$ yarn open (or npm run open)
```

### Auto-rebuild and reload on file save

Our favorite prototyping tool is [`budo`](https://github.com/mattdesl/budo):

```sh
$ yarn global add budo
```
or
```sh
$ npm install budo -g
```

```sh
$ yarn
$ yarn start
```
or
```sh
$ npm install
$ npm start
```
